# Calculating-Family-Expenses-Using-Service-Now
# Project Overview
Calculating-Family-Expenses-using-ServiceNow is a comprehensive application built on the ServiceNow platform designed to streamline the management and calculation of household financial expenditures.

This repository serves as a complete archive of the project's Software Development Life Cycle (SDLC). It contains everything from the initial ideation and empathy mapping to the final project report, including technical documentation, architectural diagrams, performance testing results, and video demonstrations of the working application.
